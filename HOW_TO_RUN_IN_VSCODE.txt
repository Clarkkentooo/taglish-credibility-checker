How to run the Tsek web app in VS Code

1. Clone the repository:
   git clone https://github.com/Clarkkentooo/taglish-credibility-checker.git

2. Open the project folder in VS Code:
   cd taglish-credibility-checker
   code .

3. Open the VS Code terminal.

4. Install dependencies:
   corepack enable
   corepack pnpm install

5. Create a local environment file if needed:
   Copy .env.example to .env.local if .env.example exists.
   The current frontend works with these defaults:
   NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
   NEXT_PUBLIC_USE_MOCK_API=true

6. Start the development server:
   corepack pnpm dev

7. Open the app in a browser:
   http://localhost:3000

Useful commands:
   corepack pnpm lint
   corepack pnpm typecheck
   corepack pnpm test
   corepack pnpm test:e2e
   corepack pnpm build

Notes:
   The app currently uses mock analysis data by default.
   Backend integration can be added later by setting NEXT_PUBLIC_USE_MOCK_API=false
   and pointing NEXT_PUBLIC_API_BASE_URL to the backend server.
